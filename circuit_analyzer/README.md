# Анализатор электрических цепей постоянного тока


## 📋 О проекте
Программный модуль для анализа электрических цепей постоянного тока на основе законов Кирхгофа. Разработан в рамках конференции по физике (Секция 8. Индустриальное программирование).

### Основные возможности
- ✅ **Расчет цепей постоянного тока** методом узловых потенциалов
- ✅ **Поддержка компонентов**: резисторы (R), источники напряжения (V)
- ✅ **Визуализация**:
  - График напряжений в узлах
  - График токов в компонентах
  - Схема цепи в графовом представлении
- ✅ **Готовые примеры** для быстрого старта
- ✅ **Сохранение и загрузка** схем в формате JSON
- ✅ **Индустриальная архитектура**: разделение на Model-View-Controller


### Установка
1.Клонируйте репозиторий:
```bash
git clone https://github.com/Helen11-png/software-module-for-the-analysis-of-DC-electrical-circuits-based-on-Kirchhoff-s-laws.git
cd circuit_analyzer 
```


2.Создайте виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate  # Windows
```

3.Установите зависимости:

```bash
pip install -r requirements.txt
```
4. Запуск
```bash
python main.py
```
